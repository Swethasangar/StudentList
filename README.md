# StudentList
Student List Project
Student List
A simple React-based application to manage a list of students, with functionality to mark favorite students and remove them from the list. The app uses React Context for state management, allowing easy access to the student list data across different components.

Features
List of Students: Displays a list of students, including the ability to favorite certain students.
Favorite Students: View a separate list of students marked as favorites.
Remove from Favorites: Allows users to remove a student from the favorite list.

Technologies Used
React: Front-end JavaScript library for building user interfaces
CSS: For styling the application
